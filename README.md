Sansa
=====

Fantasy roguelike game.

![screenshot](http://ww4.sinaimg.cn/large/62c1e01egw1etvjxleihjj20l20d30uk.jpg)


## Build and Run
Need [Rust compiler and Cargo](http://www.rust-lang.org/install.html).

    cargo run
